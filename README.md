# PalasFlix (simples)

Versão mínima para TizenBrew, sem overengineering.

## Arquivos
- `package.json`
- `service.js`
- `index.html`
- `app.js`
- `styles.css`

## Instalação
No TizenBrew, use **Install from GitHub** e informe: `Palaski/PalasFlix`.

## Sobre GitHub Pages
Não é obrigatório para módulo app do TizenBrew quando a instalação é feita por repositório GitHub.
O loader usa os arquivos do repo (manifesto + entrypoint). Então Pages desativada normalmente **não impede** o módulo de abrir.

## Se você alterou muitos arquivos
Esta branch foi normalizada para uma base mínima novamente. Reinstale o módulo no TizenBrew após atualizar o repositório.
